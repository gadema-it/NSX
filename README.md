## NSX

NSX is a prototype for a 3D modeling application in QT/C++ and OpenGL.
It uses the halfedge data structure implementing triangulation by earcutting, normal automatic calculation, subdivision using opensubdiv.

### Hotkeys

navigation: S + mouse

selection:
	 object: Spacebar 
	 vertex: V 
	 edge: C
	 face: X
	 
translate: W
rotate: E
scale: R

delete face: \`

increase/descrease subdivision: page up / page down
